#### **[ 绑定 ]集合 + 数组**

>[!WARNING|style: flat|label:  简要说明 ]
>
>当目标类型是一个集合数组 (`Key: 路径前缀 index, Value: 集合的元素 - 基元类型`)
>
>- 指定目标类型`IReadOnlyList<T>, ICollection<T>, IReadOnlyCollection<T>, IEnumerable<>`则创建`List<T>`
>
>  <span style='color:red'>[ 绑定过程中，不会因为某个元素的绑定失败而终止 - 集合忽略该元素，数组：生成默认元素 ]</span>
>
>  <br/>

```csharp
public class User
{
    
     public string Name { get; set; }
     public int Age { get; set; }
     public ContactInfo ContactInfo { get; set; }
    
}

public class ContactInfo
{
    
     public string Email { get; set; }
     public string PhontNo { get; set; }
      
}


```

>```csharp
>string @json = """
>	 [
>	      {
>	           "Name" = "张三",
>	           "Age" = "20",
>	           "ContactInfo": { "Email" = "13810000000@qq.com", "PhontNo" = "20" }
>	      },
>	      {
>	           "Name" = "李四",
>	           "Age" = "20",
>	           "ContactInfo": { "Email" = "13810000000@qq.com", "PhontNo" = "20" }
>	      }
>	 ]
>	 """;
>
>
>```
>
>```csharp
>var source = new Dictionary<string, string>
>{
>
>      ["0:Name"] = "张三",
>      ["0:Age"] = "20",
>      ["0:ContactInfo:Email"] = "13810000000@qq.com",
>      ["0:ContactInfo:PhontNo"] = "20",
>
>      ["1:Name"] = "李四",
>      ["1:Age"] = "20",
>      ["1:ContactInfo:Email"] = "13810000000@qq.com",
>      ["1:ContactInfo:PhontNo"] = "20",
>
>};
>IConfigurationRoot root = new ConfigurationBuilder()
>                              .AddInMemoryCollection(source)
>                              .Build();
>
>// [ 集合 ]
>IEnumerable<User> us = root.Get<IEnumerable<User>>();
>// [ 数组 ]
>User[] us = root.Get<User[]>();
>
>
>```

#### **[ 绑定字典 ]**

>[!WARNING|style: flat|label:  简要说明 ]
>
>当目标类型是一个字典 (`Key: 路径前缀 字符串或枚举, Value: 字典的元素 - 基元类型`)
>
>- 指定目标类型`IReadOnlyDictionary<,>, IDictionary<,>`
>
>  <span style='color:red'>[ 集合的配置绑定：不会因为某个元素的绑定失败而终止 ]</span>
>
>  <br/>

```csharp
public class User
{
    
     public string Name { get; set; }
     public int Age { get; set; }
     public ContactInfo ContactInfo { get; set; }
    
} 

public class ContactInfo
{
    
     public string Email { get; set; }
     public string PhontNo { get; set; }
     
}


```
