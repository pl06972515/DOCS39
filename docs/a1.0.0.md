>[!WARNING|style: flat|label: 简要说明 ]
>
>[`AgileConfig`]是一个基于`.Net Core`开发的轻量级配置中心 <span style='color:Red'>[ 支持`IConfiguration & IOptions`模式 ]</span>
>
>- 部署简单，支持`N`节点分布式部署(`至少需要一个数据节点`)
>
>    <span style='color:red'>[ 使用长连接技术 - 配置信息实时推送至客户端 ]</span>
>
>
> ---
>
>[<span style='color:#008B00'>[👓 AgileConfig - 架构说明 ]</span>](https://github.com/dotnetcore/AgileConfig#-%E5%8A%A1%E5%BF%85%E5%9C%A8%E4%BD%BF%E7%94%A8-agileconfig-%E4%B9%8B%E5%89%8D%E4%BB%94%E7%BB%86%E9%98%85%E8%AF%BB%E4%BB%A5%E4%B8%8B%E6%96%87%E6%A1%A3 ':target=_blank')
> 
>![image-20230104151913487](wwwroot/DocImages/image-20230104151913487.png ':size=700x300')
>
><br/>
>
>- [`客户端 - Client`]客户端程序由(`NetStandard v2.0`)开发的一个类库 
>
>- [`节点服务 - Node Server`] [`控制台管理 - Console`]节点使用`.Net Core`开发的一个服务
>
>  <span style='color:red'>[ ⚠ 简化部署`AgileConfig`并没有单独的控制台程序，使用任意一个节点作为控制台管理 ]</span> 
>
> - [`数据库 - DataBase`]使用数据库来存储数据(`目前支持 Sqlserver, Mysql, Sqlite, PostgreSql, Oracle 五种数据库`)
>
